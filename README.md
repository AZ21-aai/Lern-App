# Lern-App (IHK Abschlussprojekt)

Fullstack Webanwendung zur Prüfungsvorbereitung, entwickelt im Rahmen meiner Umschulung zum Fachinformatiker für Anwendungsentwicklung.

## 🚀 Technologien

- Frontend: React, HTML, CSS
- Backend: Java, Spring Boot (REST API)
- Datenbank: PostgreSQL

## 📚 Projektbeschreibung

Die Lern-App ermöglicht eine strukturierte Vorbereitung auf Prüfungen durch themenbasierte Multiple-Choice-Fragen. Nutzer können gezielt Themen auswählen und erhalten direkt Feedback zu ihren Antworten.

Das Projekt wurde vollständig eigenständig konzipiert und umgesetzt – von der Planung über die Datenmodellierung bis hin zur Implementierung.

## 🏗️ Architektur

- Client-Server-Architektur
- REST API Kommunikation zwischen Frontend und Backend
- Relationale Datenbankstruktur mit PostgreSQL

## 📄 Dokumentation

Die vollständige Projektdokumentation sowie die Präsentation sind im Ordner `docs` enthalten.

## ▶️ Projekt starten

### Backend starten:
`cd backend && ./gradlew bootRun`

### Frontend starten:
`cd frontend && npm install && npm start`
